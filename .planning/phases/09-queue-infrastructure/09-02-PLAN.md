**stock/queue_consumer.py:**

Module-level constants:
- `COMMAND_STREAM = "{queue}:stock:commands"`
- `REPLY_STREAM = "{queue}:replies"`
- `CONSUMER_GROUP = "stock-consumers"`
- `CONSUMER_NAME = "stock-1"`
- `POLL_INTERVAL_MS = 1000`
- `BATCH_SIZE = 10`
- `STREAM_MAXLEN = 1_000`

Import `operations` (same directory import, works because stock/ is on sys.path at runtime).
Import `msgspec.json` for payload/result serialization.

Command dispatch table:
```python
COMMAND_DISPATCH = {
    "reserve_stock": lambda db, p: operations.reserve_stock(db, p["item_id"], int(p["quantity"]), p["idempotency_key"]),
    "release_stock": lambda db, p: operations.release_stock(db, p["item_id"], int(p["quantity"]), p["idempotency_key"]),
    "check_stock": lambda db, p: operations.check_stock(db, p["item_id"]),
}
```

IMPORTANT: Cast `quantity` and `amount` to `int()` because msgspec.json may decode them as int already, but defensive casting ensures correctness regardless of serialization.

Function `setup_command_consumer_group(queue_db)`:
- Same pattern as consumers.py -- xgroup_create with mkstream=True, catch BUSYGROUP

Function `queue_consumer(db, queue_db, stop_event: asyncio.Event)`:
- `db` = service's own Redis (for operations calls)
- `queue_db` = shared queue Redis (for stream reads/writes)
- Loop pattern from consumers.py: while not stop_event.is_set()
- XREADGROUP from COMMAND_STREAM
- For each message:
  - Decode fields: `correlation_id = fields[b"correlation_id"].decode()`, `command = fields[b"command"].decode()`, `payload_str = fields[b"payload"].decode()`
  - Parse payload: `payload = msgspec.json.decode(payload_str.encode())`
  - Look up handler in COMMAND_DISPATCH. If not found: `result = {"success": False, "error_message": f"unknown command: {command}"}`
  - Else: `result = await handler(db, payload)`
  - Publish reply: `await queue_db.xadd(REPLY_STREAM, {"correlation_id": correlation_id, "result": msgspec.json.encode(result).decode()}, maxlen=STREAM_MAXLEN, approximate=True)`
  - ACK: `await queue_db.xack(COMMAND_STREAM, CONSUMER_GROUP, msg_id)`
- Wrap xreadgroup in try/except: CancelledError re-raise, others log warning + sleep(1)
- Log info on start/stop
