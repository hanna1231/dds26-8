Create `orchestrator/saga.py` with:

**1. SAGA hash field schema** (key: `saga:{order_id}`):
- `state`: one of STARTED, STOCK_RESERVED, PAYMENT_CHARGED, COMPLETED, COMPENSATING, FAILED
- `order_id`, `user_id`, `total_cost`: string copies from CheckoutRequest
- `items_json`: JSON-encoded list of {"item_id": str, "quantity": int} dicts (from repeated LineItem)
- `stock_reserved`: "0" or "1" — set to "1" after all ReserveStock calls succeed
- `payment_charged`: "0" or "1" — set to "1" after ChargePayment succeeds
- `refund_done`: "0" or "1" — set to "1" after RefundPayment compensation succeeds
- `stock_restored`: "0" or "1" — set to "1" after all ReleaseStock compensations succeed
- `error_message`: string, set on failure
- `started_at`, `updated_at`: unix timestamps as strings

**2. Valid state transitions table** (dict):
```python
VALID_TRANSITIONS = {
    "STARTED": {"STOCK_RESERVED", "COMPENSATING"},
    "STOCK_RESERVED": {"PAYMENT_CHARGED", "COMPENSATING"},
    "PAYMENT_CHARGED": {"COMPLETED", "COMPENSATING"},
    "COMPENSATING": {"FAILED"},
}
```

**3. TRANSITION_LUA** — Lua CAS script for atomic state transition:
```lua
local current = redis.call('HGET', KEYS[1], 'state')
if current ~= ARGV[1] then return 0 end
redis.call('HSET', KEYS[1], 'state', ARGV[2])
redis.call('HSET', KEYS[1], 'updated_at', tostring(math.floor(redis.call('TIME')[1])))
if ARGV[3] ~= '' then redis.call('HSET', KEYS[1], ARGV[3], ARGV[4]) end
return 1
```

**4. Functions:**

`create_saga_record(db, order_id, user_id, items, total_cost) -> bool`:
- Use `HSETNX` on `state` field to atomically prevent duplicate SAGA creation (Pitfall 3 from research).
- If HSETNX returns 1 (created), HSET all remaining fields and set EXPIRE 7 days.
- If HSETNX returns 0 (already exists), return False.
- `items` is a list of dicts [{"item_id": str, "quantity": int}]; store as JSON string in `items_json` field.

`transition_state(db, saga_key, from_state, to_state, flag_field="", flag_value="") -> bool`:
- Validate transition against VALID_TRANSITIONS dict before calling Lua (fail fast on invalid).
- Call `db.eval(TRANSITION_LUA, 1, saga_key, from_state, to_state, flag_field, flag_value)`.
- Return True if Lua returned 1, False otherwise.

`get_saga(db, order_id) -> dict | None`:
- `db.hgetall(f"saga:{order_id}")` then decode bytes: `{k.decode(): v.decode() for k, v in raw.items()}` (Pitfall 4).
- Return None if hash is empty.

`set_saga_error(db, order_id, error_message)`:
- `db.hset(f"saga:{order_id}", "error_message", error_message)`.

All functions are async. Use `import time` for timestamps. Do NOT import from app.py to avoid circular imports. Do NOT use `decode_responses=True` on the Redis client — decode manually in get_saga (consistent with existing codebase pattern in grpc_server.py).
