1. **Add run_2pc_checkout to orchestrator/grpc_server.py**:

   ```python
   import os

   ```

   Implement `run_2pc_checkout(db, order_id, user_id, items, total_cost) -> dict`:
   - **Exactly-once check**: Call get_tpc(db, order_id). If exists: COMMITTED -> return success, ABORTED -> return failure, other -> return "checkout already in progress".
   - **Create TPC record**: Call create_tpc_record. Handle race (same pattern as SAGA).
   - **Phase 1 (INIT -> PREPARING)**: transition_tpc_state(db, tpc_key, "INIT", "PREPARING")
   - **Concurrent PREPARE via asyncio.gather**: Build futures list with prepare_stock(item_id, quantity, order_id) for each item, plus prepare_payment(user_id, total_cost, order_id). Call asyncio.gather(*futures, return_exceptions=True).
   - **Collect votes**: Check each result. If isinstance(r, Exception) or not r.get("success"), that's a NO vote.
   - **If all YES**:
     - WAL: transition_tpc_state(db, tpc_key, "PREPARING", "COMMITTING") -- MUST be before commits
     - Send COMMITs: asyncio.gather(commit_stock(item_id, order_id) for each item, commit_payment(user_id, order_id))
     - Finalize: transition_tpc_state(db, tpc_key, "COMMITTING", "COMMITTED")
     - Return {"success": True, "error_message": ""}
   - **If any NO**:
     - WAL: transition_tpc_state(db, tpc_key, "PREPARING", "ABORTING") -- MUST be before aborts
     - Send ABORTs: asyncio.gather(abort_stock(item_id, order_id) for each item, abort_payment(user_id, order_id), return_exceptions=True) -- use return_exceptions so one abort failure doesn't prevent others
     - Finalize: transition_tpc_state(db, tpc_key, "ABORTING", "ABORTED")
     - Extract first error from failed results
     - Return {"success": False, "error_message": error_msg}

   **Update StartCheckout** to route based on TRANSACTION_PATTERN:
   ```python
   async def StartCheckout(self, request, context):
       items = [{"item_id": item.item_id, "quantity": item.quantity} for item in request.items]
       if TRANSACTION_PATTERN == "2pc":
           result = await run_2pc_checkout(self.db, request.order_id, request.user_id, items, request.total_cost)
       else:
           result = await run_checkout(self.db, request.order_id, request.user_id, items, request.total_cost)
       return CheckoutResponse(success=result["success"], error_message=result["error_message"])
   ```
