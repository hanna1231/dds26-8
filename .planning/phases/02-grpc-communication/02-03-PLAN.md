**1. Create `orchestrator/client.py`:**

Module-level globals for channels and stubs (per locked decision: reuse channels, no per-call creation):

```python
import os
import grpc.aio
from stock_pb2 import ReserveStockRequest, ReleaseStockRequest, CheckStockRequest
from stock_pb2_grpc import StockServiceStub
from payment_pb2 import ChargePaymentRequest, RefundPaymentRequest, CheckPaymentRequest
from payment_pb2_grpc import PaymentServiceStub

_stock_channel = None
_payment_channel = None
_stock_stub: StockServiceStub = None
_payment_stub: PaymentServiceStub = None

RPC_TIMEOUT = 5.0  # seconds, per locked decision

STOCK_ADDR = os.environ.get("STOCK_GRPC_ADDR", "stock-service:50051")
PAYMENT_ADDR = os.environ.get("PAYMENT_GRPC_ADDR", "payment-service:50051")
```

**init_grpc_clients(stock_addr=None, payment_addr=None):**
- Create `grpc.aio.insecure_channel(addr)` for each service
- Create stubs from channels
- Allow override of addresses via parameters (for testing) falling back to env vars

**close_grpc_clients():**
- Close both channels if they exist

**Wrapper functions** — each returns a dict with `success` and `error_message` (plus extra fields for check operations). Per locked decision: all mutation functions accept `idempotency_key` parameter:

- `async def reserve_stock(item_id: str, quantity: int, idempotency_key: str) -> dict`
  - Calls `_stock_stub.ReserveStock(ReserveStockRequest(...), timeout=RPC_TIMEOUT)`
  - Returns `{"success": resp.success, "error_message": resp.error_message}`

- `async def release_stock(item_id: str, quantity: int, idempotency_key: str) -> dict`
  - Calls `_stock_stub.ReleaseStock(ReleaseStockRequest(...), timeout=RPC_TIMEOUT)`
  - Returns same format

- `async def check_stock(item_id: str) -> dict`
  - Calls `_stock_stub.CheckStock(CheckStockRequest(item_id=item_id), timeout=RPC_TIMEOUT)`
  - Returns `{"success": resp.success, "error_message": resp.error_message, "stock": resp.stock, "price": resp.price}`

- `async def charge_payment(user_id: str, amount: int, idempotency_key: str) -> dict`
  - Calls `_payment_stub.ChargePayment(ChargePaymentRequest(...), timeout=RPC_TIMEOUT)`
  - Returns `{"success": resp.success, "error_message": resp.error_message}`

- `async def refund_payment(user_id: str, amount: int, idempotency_key: str) -> dict`
  - Calls `_payment_stub.RefundPayment(RefundPaymentRequest(...), timeout=RPC_TIMEOUT)`
  - Returns same format

- `async def check_payment(user_id: str) -> dict`
  - Calls `_payment_stub.CheckPayment(CheckPaymentRequest(user_id=user_id), timeout=RPC_TIMEOUT)`
  - Returns `{"success": resp.success, "error_message": resp.error_message, "credit": resp.credit}`

**No error wrapping in this module** — let gRPC exceptions (grpc.aio.AioRpcError) propagate to the caller. Phase 4 adds circuit breaker handling. The caller (Phase 3 SAGA orchestrator) will handle transport errors.
