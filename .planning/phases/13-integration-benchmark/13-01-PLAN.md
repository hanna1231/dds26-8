from transport import COMM_MODE

@app.before_serving
async def startup():

        from reply_listener import setup_reply_consumer_group, reply_listener

**stock/app.py changes:**
1. Add `import asyncio` and `import os` at top (os already imported, asyncio is new).
2. Add module-level `COMM_MODE = os.environ.get("COMM_MODE", "grpc")` and `_stop_event = None`.
3. In `startup()`, add `global _stop_event` alongside existing `global db`.
4. After the existing `app.add_background_task(serve_grpc, db)` line, add COMM_MODE-conditional block:
   ```python
   if COMM_MODE == "queue":
       from queue_consumer import setup_command_consumer_group, queue_consumer
       _stop_event = asyncio.Event()
       await setup_command_consumer_group(db)
       app.add_background_task(queue_consumer, db, db, _stop_event)
   ```
   Note: Pass `db` for BOTH params (db and queue_db) because in simple mode they share the same Redis cluster.
5. In `shutdown()`, add `if _stop_event: _stop_event.set()` BEFORE the existing `await stop_grpc_server()` line.
