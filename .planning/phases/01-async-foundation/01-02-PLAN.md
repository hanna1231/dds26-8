Migrate stock/app.py from Flask to Quart with async Redis. Apply these specific changes:

**Import swap:**
- `from flask import Flask, jsonify, abort, Response` -> `from quart import Quart, jsonify, abort, Response`
- `import redis` -> `import redis.asyncio as redis`
- Remove `import atexit`
- `app = Flask("stock-service")` -> `app = Quart("stock-service")`

**Redis client lifecycle (replace module-level db and atexit):**
- Remove the module-level `db: redis.Redis = redis.Redis(...)` initialization
- Remove the `close_db_connection()` function and `atexit.register(close_db_connection)`
- Declare `db: redis.Redis = None` as module-level placeholder
- Add `@app.before_serving` async startup hook:
  ```python
  @app.before_serving
  async def startup():
      global db
      db = redis.Redis(host=os.environ['REDIS_HOST'],
                       port=int(os.environ['REDIS_PORT']),
                       password=os.environ['REDIS_PASSWORD'],
                       db=int(os.environ['REDIS_DB']))
  ```
- Add `@app.after_serving` async shutdown hook:
  ```python
  @app.after_serving
  async def shutdown():
      await db.aclose()
  ```

**Make all route handlers and helpers async:**
- `get_item_from_db` -> `async def get_item_from_db` with `await db.get(item_id)`
- `create_item` -> `async def create_item` with `await db.set(key, value)`
- `batch_init_users` -> `async def batch_init_users` with `await db.mset(kv_pairs)`
- `find_item` -> `async def find_item` with `await get_item_from_db(item_id)`
- `add_stock` -> `async def add_stock` with `await get_item_from_db(item_id)` and `await db.set(...)`
- `remove_stock` -> `async def remove_stock` with `await get_item_from_db(item_id)` and `await db.set(...)`

**Preserve abort() calls exactly as-is.** `abort()` does NOT need `await`. Keep `return abort(400, DB_ERROR_STR)` patterns unchanged.

**Update __main__ and remove gunicorn logger block:**
- Replace `if __name__ == '__main__': app.run(host="0.0.0.0", port=8000, debug=True)` with:
  ```python
  if __name__ == '__main__':
      app.run(host="0.0.0.0", port=5000, debug=True)
  ```
- Remove the entire `else:` block (gunicorn_logger wiring).

**Do NOT change:** StockValue Struct, msgspec imports, msgpack encode/decode calls, route paths, response formats, error strings, any business logic.
